# Issue backlog seeds

Use these starter issues to make the repository easier for new contributors to enter.

## Good first issues

### Add artifact schema validator CLI

Labels: `enhancement`, `good first issue`, `python`

Build a command that validates generated JSON artifacts under an artifact root and reports missing files, invalid JSON, and basic schema gaps.

### Add CSV export for composite OHLCV artifacts

Labels: `enhancement`, `good first issue`, `python`

Export `composite_ohlcv*.json` into a flat CSV file suitable for spreadsheet, DuckDB, or pandas users.

### Improve Windows PowerShell quickstart

Labels: `documentation`, `good first issue`

Add venv-safe PowerShell commands that do not require changing execution policy.

## Help wanted issues

### Add Coinbase public market-data connector

Labels: `enhancement`, `help wanted`, `python`

Add a public REST connector with mocked parser tests. Do not add private account APIs or trading endpoints.

### Add static HTML artifact dashboard frontend

Labels: `enhancement`, `help wanted`, `dashboard`

Build a small static UI on top of the existing read-only dashboard API. It should show artifact health and data-quality state, not trading signals.
